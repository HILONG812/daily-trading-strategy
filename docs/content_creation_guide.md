# AI 自媒体内容创作指南

## 小红书创作

### 内容定位
- 投资笔记分享
- 交易心得
- 市场分析
- 学习成长

### 标题模板
1. "月薪 X 千如何理财？我的实盘记录"
2. "AI 帮我炒股，3 个月收益 X%"
3. "普通人如何建立投资体系"
4. "今天又亏了，但我想说..."

### 内容结构
```
标题（吸引眼球）
↓
开头（痛点/共鸣）
↓
正文（干货 + 数据）
↓
结尾（互动引导）
↓
标签（#投资 #理财 #AI）
```

### 注意事项
- 不要承诺收益
- 注明"模拟交易，不构成投资建议"
- 真实记录盈亏
- 配图：交易截图 + 文字说明

## 微信公众号/订阅号

### 内容类型
1. **日报**：每日市场复盘
2. **周报**：周度总结 + 下周展望
3. **深度**：行业分析、公司研究
4. **心得**：投资哲学、方法论

### 排版规范
```
标题（20 字以内）
↓
导语（50 字概括）
↓
正文（分段 + 小标题）
↓
总结（核心观点）
↓
免责声明
```

### 工具推荐
- 排版：秀米、135 编辑器
- 配图：Canva、稿定设计
- 数据：TradingView、Yahoo Finance
