# 运营专员工作手册

## 📋 职责范围

1. **飞书推送内容优化** - 每日/每周报告推送
2. **用户增长策略** - 待实施
3. **内容运营** - 交易日报、周报
4. **数据分析** - 胜率、收益率统计
5. **社群运营** - 如有需要

---

## 📊 工作节奏

### 每日任务 (交易日)
- [ ] 盘后更新交易数据 (trades.json)
- [ ] 生成交易日报 (operations/daily-reports/YYYY-MM-DD.md)
- [ ] 推送飞书日报 (含关键词"老板")
- [ ] 更新分析看板 (operations/analytics/dashboard.md)
- [ ] 记录当日交易逻辑和心得

### 每周任务 (周五/周末)
- [ ] 生成本周周报 (operations/weekly-reports/YYYY-Www.md)
- [ ] 推送飞书周报
- [ ] 更新周度统计数据
- [ ] 制定下周交易计划
- [ ] 复盘本周策略执行

### 每月任务 (月末)
- [ ] 月度收益统计
- [ ] 策略有效性分析
- [ ] 下月目标设定
- [ ] 配置优化建议

## 📈 核心指标追踪

### 收益指标
- 总收益 (绝对值 + 百分比)
- 日均收益
- 年化收益率
- 月度收益

### 胜率指标
- 总交易次数
- 盈利交易数
- 亏损交易数
- 胜率 (盈利/总交易)
- 盈亏比 (平均盈利/平均亏损)

### 风险指标
- 最大回撤
- 仓位使用率
- 单一市场占比
- 单一股票占比
- 行业集中度

## 🎯 报告模板

### 日报结构
1. 账户概览 (资金、仓位、交易数)
2. 持仓明细 (按市场分类)
3. 今日交易记录
4. 配置分析 (地域、行业)
5. 风险提示
6. 明日关注
7. 策略备注

### 周报结构
1. 本周概览 (收益、交易数、胜率)
2. 持仓表现 (建仓明细)
3. 配置分析 (图表展示)
4. 本周交易记录
5. 策略执行复盘
6. 市场回顾
7. 经验总结
8. 下周计划
9. 下周目标

## 📌 飞书推送规范

### 关键词
必须包含 "老板" 才能通过 webhook 验证

### 推送时间
- 日报：交易日 16:30-18:00 (美东时间盘后)
- 周报：周五 18:00 后或周末

### 内容格式
- 简洁版：飞书消息 (300-500 字)
- 详细版：飞书文档/Markdown 文件

### 消息结构
1. 称呼 + 日期
2. 核心数据 (3-5 个关键指标)
3. 重点持仓/交易
4. 风险提示
5. 下一步计划
