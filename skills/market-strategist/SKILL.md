# 市场策略师技能手册

## 角色定位

全球市场策略师，融合达利奥系统逻辑 + 芒格多元思维模型 + K 线技术分析

---

## 核心思维框架

### 达利奥思维
- 市场是一台"机器"，看底层经济力量
- 债务周期、通胀、增长、地缘政治
- 分析必须系统化、可回测

### 芒格思维
- 多元思维模型
- 寻找具备"护城河"的企业
- 逆向思维发现风险
- 警惕人类误判心理学

### 技术分析
- 日本蜡烛图技术
- K 线形态捕捉市场情绪转折点
- 关键形态：吞没、启明星、三只乌鸦、锤子线

---

## 分析能力

### 技术指标
- RSI(14)：超买>70，超卖<30
- MACD：趋势动能
- 布林带：波动范围
- 均线交叉：MA5/20/60

### 趋势判断
- 站上均线 = 多头
- 跌破均线 = 空头
- RSI 背离 = 反转信号

### 基本面分析
- 财报关键指标：EPS、Revenue、PE、ROE、负债率
- 行业分析：市场规模、增长率、竞争格局
- 护城河分析：品牌、渠道、网络效应、成本优势

### 估值模型
- DCF 折现现金流：未来现金流折现到现值
- PE 市盈率：股价/EPS，<20 算便宜
- PB 市净率：股价/每股净资产，<1 算便宜
- PEG：PE/增长率，<1 被低估

## 风险管理

### 凯利公式
f = (bp - q) / b
- f = 应投资比例
- b = 赔率
- p = 胜率
- q = 败率 (1-p)

### 分散投资
- 不同行业
- 不同市场 (美/港/日)

### 最大回撤
- 控制在 20% 以内

**推荐方案**: 
1. 模拟交易：雪球/富途模拟盘
2. 实盘交易：币安 API (加密货币) 或 富途 SDK (股票)

**核心逻辑**:
```python
# 1. 登录
page.goto('https://example-trading.com/login')
page.fill('#username', credentials['user'])
page.fill('#password', credentials['pass'])
page.click('#login-btn')

# 2. 监控行情 (每 5 分钟)
async def monitor_price():
    while True:
        await page.reload()
        price = await page.text_content('.price-element')
        
        # 3. 条件触发
        if float(price) < threshold:
            await page.click('#buy-btn')
            log_trade('BUY', price)
        
        await asyncio.sleep(300)  # 5 分钟
```

- 止盈：+20% 减半仓
