## 一、巴菲特 (Warren Buffett) - 价值投资

### 核心理念

**1. 护城河 (Moat)**
- 品牌护城河：可口可乐、苹果
- 成本护城河：Costco、好市多
- 网络效应护城河：亚马逊、信用卡
- 政府监管护城河：公用事业、保险公司

**2. 安全边际**
- 内在价值 = 5-7年后自由现金流的折现
- 买入价 < 内在价值的 50%
- "别人贪婪时恐惧，别人恐惧时贪婪"

**3. 长期持有**
- "如果你不想持有10年，就不要持有10分钟"
- 复利的力量：20%年化，10年6倍，20年38倍
- 减少交易频率 = 减少摩擦成本

**4. 能力圈**
- 只投资自己看得懂的生意
- "对你的能力圈来说，最重要的不是能力圈的大小，而是知道你能力圈的边界在哪里"

**5. 管理层**
- 芒格："找一位聪明、诚实、有能力的人，然后把权力下放给他"
- 关注管理层如何分配资本

### 巴菲特指标

| 指标 | 含义 | 应用 |
|------|------|------|
| PE (市盈率) | 价格/盈利 | 越低越好，但要看行业 |
| ROE (净资产收益率) | 盈利能力 | >15% 算优秀 |
| 自由现金流 | 真正能拿回家的钱 | 持续为正 |
| 格雷厄姆数字 | 内在价值估算 | PE×PE×EPS×1.5 |

## 二、芒格 (Charlie Munger) - 多元思维模型

### 核心思维

**1. 逆向思考**
- "如果我知道我会死在哪里，我就永远不去那里"
- 想成功？先研究失败案例

**2. 100个思维模型**
- 物理学：临界点、均衡
- 生物学：进化、竞争
- 心理学：误判心理、激励机制
- 经济学：规模效应、供需曲线

**3. Lollapalooza 效应**
- 多个因素同时起作用时的放大效果
- 往往比单一因素重要得多

**4. 误判心理学**
- 奖励/惩罚超级反应
- 喜欢/热爱倾向
- 讨厌/憎恨倾向
- 避免怀疑/避免不一致
- 社会认同
- 对比效应

**5. 能力圈 + 竞争优势**
- "钓鱼要在鱼多的地方钓"
- 等待好球再挥棒

## 三、达利欧 (Ray Dalio) - 原则

### 经济机器运行规律

**1. 债务周期**
- 短期债务周期：7-10年
- 长期债务周期：50-75年
- 去杠杆的4种方式：压缩支出、债务重组、债务货币化、财富转移

**2. 三大均衡**
- 债务增长 = 收入增长
- 生产率提升
- 短期/长期债务周期

**3. 原则清单**

| 领域 | 核心原则 |
|------|----------|
| 生活 | 拥抱现实、极度开放 |
| 工作 | 可信度加权、创意择优 |
| 投资 | 分散化、投资回报相关系数低 |

**4. 投资圣杯**
- 找到15个以上相关性低的好资产
- 风险平价 (Risk Parity)

## 四、日本蜡烛图 (Steve Nison)

### 基础形态

**1. 单根K线**
| 形态 | 含义 |
|------|------|
| 锤子线 (Hammer) | 底部反转，看涨 |
| 上吊线 (Hanging Man) | 顶部反转，看跌 |
| 流星线 (Shooting Star) | 顶部信号，看跌 |
| 倒锤子线 (Inverted Hammer) | 底部信号，看涨 |

**2. 吞没形态 (Engulfing)**
- 阳包阴：看涨
- 阴包阳：看跌
- 需配合趋势

**3. 孕线 (Harami)**
- 十字孕线是强烈反转信号
- "母亲包容孩子"

**4. 十字星 (Doji)**
- 开=收，买卖平衡
- 长期十字星 = 趋势暂停/反转

### 持续形态

| 形态 | 信号 |
|------|------|
| 三法 (Three Methods) | 趋势延续 |
| 上升/下降三法 | 继续沿原趋势 |
| 窗口 (Gap) | 趋势加速 |

### 实战要点

1. **趋势第一**
   - 只在趋势明确时交易
   - 逆势操作需要更强信号

2. **位置比形态重要**
   - 顶部大阳线 = 可能是出货
   - 底部大阳线 = 可能是启动

3. **成交量确认**
   - 放量突破 = 真突破
   - 缩量回调 = 机会

4. **均线配合**
   - 站稳20日均线 = 多头
   - 跌破20日均线 = 观望

## 五、综合实战框架

### 选股逻辑

```
1. 行业格局好 (有护城河)
2. 公司壁垒强 (定价权)
3. 管理层靠谱 (资本配置)
4. 财务健康 (现金流、负债)
5. 估值合理 (安全边际)
```

### 技术面确认

```
1. 月线趋势向上
2. 周线调整结束
3. 日线出现买入形态
4. 带量突破关键位
```

### 仓位管理

```
- 单只股票 < 10%
- 行业 < 20%
- 现金/债券 >= 20%
- 保留应对极端情况
```
